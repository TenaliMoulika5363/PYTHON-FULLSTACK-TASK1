# User Management System

A simple User Management System built using Flask, SQLite, HTML, CSS, and Bootstrap.

## Features

- Add User
- Store User Data in SQLite Database
- Display Users in Table
- Responsive UI
- Form Validation

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap

## Project Structure

project/
│
├── app.py
├── create_db.py
├── delete_users.py
├── database.db
├── templates/
│   └── index.html
└── static/
    └── style.css

## How to Run

1. Install Flask

pip install flask

2. Create Database

python create_db.py

3. Run Application

python app.py

4. Open Browser

http://127.0.0.1:5000
